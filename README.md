# Experience Cloud Site Template
This folder contains an example Experience Cloud site that can serve as a starting point for building a checkout page using Experience Cloud and FinDock. The site is a minimal starting point for a straightforward checkout page. The flow templates can be found in the [flow-templates](https://github.com/FinDockLabs/experience-cloud-flow-template-checkout) repo and can be added to the site to create a checkout flow. The site is meant to be customized and extended to fit the specific use case and requirements of your page.

## Installation
1. Create a new Experience Cloud site with the "Build your own (LWR)" template and the name "Checkout Page".
2. Press the buttons to deploy the repo to the org.
3. Customize the page in the Experience Cloud builder
4. Publish the Experience Cloud
